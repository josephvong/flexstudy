子元素属性

子容器 属性 flex

flex 为组合属性，由flex: flex-grow flex-shrink flex-basis 组成 这三个的默认值为 "flex: 0 1 auto"

子元素等比分布

flex:0 1 auto
flex:0 1 auto
flex:0 1 auto

子容器 属性 flex-grow

flex-grow：子节点相对父容器空间扩展时所占的比例，值>=0, 当父容器 宽度增加时， 子元素会根据所有兄弟子节点 的flex-grow属性值的总和 和自身的 flex-grow 属性值 的比例进行伸展

设置父节点宽度为屏幕宽，第一个子元素固定宽度100px，后面三个分别占屏幕剩余宽度的1份 2份 3份，当屏幕宽度变化时，子节点等比变化

width:100px
flex-grow:1 (flex:1 1 auto)
flex-grow:2 (flex:2 1 auto)
flex-grow:3 (flex:3 1 auto)

兼容性 ：box-flex / -webkit-box-flex 可以 作为 flex-grow 的旧版兼容

子容器 属性 flex-shrink

flex-shrink 时flex的第二个值，值>=0，默认=1，表示当父容器空间被压缩时，子节点的压缩比例。flex-shrink=0时不压缩，shrink=1时等比压缩

设置父节点宽度为屏幕宽，第一个用“flex-basis=300px”定义固定宽度，并且定义了flex-shrink=0,后两个1:1等分剩余空间，当浏览器宽度压缩时，第二个子元素flex-shrink=3，压缩比为3。第三个子元素flex-shrink=1，压缩比为1

shrink:0 (flex:0 0 300px)
shrink:3 (flex:1 3 auto)
shrink:1 (flex:1 1 auto)

子容器属性 flex-basis

flex 属性的第三个值 ，用来定义子元素的初始宽高度

子元素 设定grow 和 shrink 均为 0，并设定 basis 为固定值，相当于给子元素设置了固定宽高度，不随父容器大小变化改变

flex:0 0 300px
0 0 100px
0 0 100px

子节点 定义了 flex-basis=300px, flex-grow=0 flex-shrink=1 ,此时子元素的默认长度为300，且最大宽度为300，当父容器大小小于所有子节点宽度之和时，flex-shrink=1控制子节点等比缩小

flex:0 1 300
flex:0 1 300
flex:0 1 300

子节点 定义了 flex-basis=200px, flex-grow=1 flex-shrink=0 , 此时子元素的最小宽度为200（shrink=0不允许缩小），当父容器宽度大于所有子节点最小宽度和时，子节点根据grow的值进行等比伸展填充父容器的空间。在父容器缩小时，子节点最小索道flex-basis 的值

flex:1 0 200
flex:1 0 200
flex:1 0 200
